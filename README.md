Bite

Bite is a native Android application developed as part of an academic Android development project.  
The app allows users to explore kitchens, dish categories, individual dishes, and add personal ratings and comments.

The project focuses on **clean architecture**, **MVVM**, **Room database**, and **Kotlin Coroutines**, with an emphasis on maintainable code and modern UI/UX.

---

## Features

- **Home Screen**
  - Kitchen cards with horizontal dish previews
  - Direct shortcut from a dish preview to its dish details screen
  - “Explore” button to navigate to dish categories

- **Dish Categories**
  - Filtered by the selected kitchen

- **Dishes List**
  - Displays all dishes under a selected category

- **Dish Details**
  - Dish image, description, and reviews count

- **Comments & Ratings**
  - Add, update, and delete personal comments
  - Star-based rating system (1–5)
  - Reviews stored locally using Room
  - Automatic synchronization of reviews count

- **Lottie Animation**
  - Success animation after adding or updating a comment
